    关于IT培训机构的个人看法

    2018-10-23
    0

    1.前言

    缘分与巧合，最近接触比较多的培训机构出来的人，以及看过关于培训机构的文章和问答。虽然没在培训机构上过课，但是接触过很多培训机构出来的人，也看过一些培训机构的课程。关于培训机构，我也有自己的看法。经历了这些，我写了这篇文章，不吹不黑的写下我对培训机构的个人见解。这篇文章可能会有些争议，如果有不同的看法，大家可以评论，但是发言望大家理性，客观，不要人身攻击！

    PS：
    1.我是一个web前端开发者，接触的人也是web前端居多，所以下面文章，提到的培训机构可能主要是针对web前端的！
    2.文章中，有小部分关于培训机构的内容，是我从朋友或者网友那里了解的，自己并没有亲身体会过，了解过，确定过。针对这种情况，在文中我会说明。

    2.关于培训机构的价值

    关于培训机构的价值，个人总结几点吧！

    第一，导师作用，告诉你应该学习什么，在困惑的时候能够给解惑。

    第二，实践作用，带着学员做项目，有实践的机会。

    第三，人脉作用，在培训班能认识很多人，有一个人脉关系。

    上面三个是我个人的一个总结，至于价值最大化到什么程度，能实现什么样的效果，有什么成果，就得看培训机构以及学员的造化了！

    3.关于培训机构的套路

    看培训机构的公开课，网上了解，以及和培训机构的学员聊天这些经历，我总结下，培训机构的一些招生套路！

    1.挂各种高大上的牌子，各种认证企业，上市企业，也有说讲师获得什么证书的等等。

    2.各种名企的合作，美团，新浪，搜狐等什么的挂一大堆大厂的牌子。

    3.包就业，很多培训机构都是打着包就业的旗号，有些甚至还说可以保证就业的工作薪水是多少以上的，比如8K，15K以上的。"来XXX学习几个月，月薪上万"、"来XXX，让您找到满意的公司"、"找不到工作,全额退款"等。这些话，想必大家多多少少都有见到过，就是一个套路，一个营销手段而已。

    4.就业后薪水直接往高里去吹，去捧，8K，10K，15K，甚至20K以上。可能从培训机构出来的人，有部分人能拿到10K或者20K以上的薪水，但是多少人能拿到？出来多久之后拿到？100人里面有多少人拿到？

    5.在招聘网站上招生，大家在网上投简历的时候，相信很多人都遇到过一些公司的招聘，要求很低，待遇也不算差。等去实地面试的时候。发现是培训机构，先劝你交多少培训费，然后保证学完之后工资能得到多少。这个其实我挺反感的，妨碍我找工作，去面试也浪费我时间！

    6.公开课洗脑（可能用洗脑这个词，有点太过了，但是大家懂我什么意思就好）。很多培训机构，都有在网上开一些公开课，试听课，免费课程等等。但是讲的东西，一半是讲课程的内容，一半可以说是洗脑，就是说各种话，让你交钱去参加培训！

    7.和一些培训完成后的人聊过，发现有些培训机构还有一个类似面试辅导这样的课程，起初我听着以为是一个面试的回答技巧，但是后来发现，根本不是我想的那样！所谓的面试辅导，就是教学员怎么去撒谎（比如吹嘘自己各种精通），在简历上怎么做假（比如自带两三年工作经验）。但是大家可以理性想下，如果真的是技术人员参与面试，觉得有多大把握能忽悠得了？

    4.看法与感受

    1.对培训机构的看法

    在文章开头的时候我说过，我接触过很多培训机构出来的人，也看过一些培训机构的课程，也有自己的看法。至于看法，我从前年年末到去年年末这段时间，在Q群里和一些聊的时候就产生了。
    过程大概是，隔三差五的就有一个从培训机构的人在群里求职。吹自己什么都会，各种精通，还自带两三年工作经验等行为，结果一问三不知！看到这些人，我实在忍不住的时候，我就出题，出的都是下面这道题，没一个能答出来的。有些还反驳说：工作上你写这些代码的？后来的过程就不说了！

    alert(a)
a();
var a=3;
function a(){
    alert(10)
}   
alert(a)
a=6;
a();

    1.关于培训机构出来的人

    我不反感培训机构出来的人。但从培训机构出来的人，我遇到过的，大部分是基础还没打牢，就说自己各种精通，自带两三年工作经验。我很反感，这样一点都不脚踏实地！

    2.关于培训机构

    我也不反感培训机构。毕竟培训机构也是为利益而生，而且我刚学js的时候，也有看过培训机构的视频，但是我反感培训机构各种虚张声势般的招生手段，营销手段。在招聘网站上发招聘信息，实际招生；有些培训机构的导师或者助教居然是那些已经培训完了，但找不到工作的人担任；有些讲师面对学员的提问连回答都不意愿回答，直接让别人到网上找答案；甚至还有些所谓的教学就是让学员看视频；教学员撒谎般的包装简历等。这样一点都不负责任！

    3.关于社区和周围的看法

    现在，在网上的社区的讨论里面，对培训机构的看法也是各种各样，但是我发现的是，无论是社区，网友还是我认识的人。大部分都对培训机构很反感，其中还包括一些从培训机构出来的人。如果培训机构和学员都很好，市场怎么会有那么多的不认可？社区怎么会有那么多的不接受？怎么会有那么多公司不想要培训班出来的人？我觉得，要被认可，首先是学员要努力学，培训机构也要负责任！

    4.关于培训机构现状

    以web前端这个行业为例子，说下培训机构的现状，关于web前端，感觉现在就是供过于求了（附上智联2017.10.11的排行榜）。我也加入了几个关于web前端招聘的QQ群，在15年的时候，招聘满天飞，这两年招聘少之又少，基本是一两个月也不见一条招聘。造成这样的情况，培训机构的功劳无疑是最大的。在我了解过的培训机构里面，有良心的屈指可数。大多数都觉得挺无良，挺坑的，使用各种虚张声势的手段招生。培训时间，很多是4个月，这样使得一年里面行业至少增加十几万甚至几十万人！我也想知道，几个月的培训时间，能培养出几个像广告所说的那样，能有那样的实力或者待遇的学员？在这几个月的时间里，学员是怎么拥有两三年工作经验的？现在web前端人从事这么多，但是两极分化比我想象中严重，一般是是公司看不上求职者，也会出现求职者看不上公司，就导致了很多学习web前端的人找不到工作，一部分公司招不到满意的web前端！

    2.对培训机构的感受

    一.首先，跟大家分享下我一个朋友的经历

    他是一个外包公司的hr。去年挑简历招人的时候，他看到很多简历上写着各种精通，项目经验也丰富，也有工作经验，他就打电话约面试时间，约了很多人（了解到这些求职者多数是培训机构的），经常出现下面的情况。

    1.约了一些人来面试，结果被被面试官批，说挑简历不认真，专门挑一些不合格的人来面试，浪费了他的时间！

    2.经常被放鸽子，约好了时间，面试爽约，入职爽约的都有，也不告诉他一下，导致锅还是他背。（在他那里了解到这个情况不止于培训出来的求职者，自学和科班的也是同样的多。大家要引以为戒，如果有什么情况，告知下，不然hr可能会遭殃）

    3.最严重的就是学历和工作经验造假（在他那里了解到，学历造假这个情况多见于培训出来的求职者，自学其次，科班的也有几个大专谎称本科的。工作经验造假也多见于培训出来的求职者，自学和科班的求职者也有，但是没那么多），学历公司上网一查就查出真假了，根本骗不了。工作经验，即使跟别人串通好，但是面试官如果多问几句，穿帮的概率很大。只要被发现造假，基本没戏，然后背锅的又是他。

    自那时候开始，他找简历的时候没有优先找那些简历了！他说不想老是被骂。

    说这个的意图，一来当个反面教材，让大家引以为鉴。二来，替我那个朋友不值。三来举例说明下面这个耐人寻味的情况：去培训机构学习，无论是简历上，经历上，给人的印象应该是一个加分项，但是现在大部分情况是一个扣分项。造成这样情况，谁之过？

    二.然后，总结一下我的感受

    1.培训机构出来的同学的简历很多类似的，简历做得很丰富，经验，经历都很足，但是实际上很多项目经历都有点虚，很多还是了解比较浅，虽然不是一问三不知的情况，但是自己的技术水平远远没能胜任自己开的工资。

    2.极端的就是上面提到的情况：学历、工作经验、项目经历的做假现象，简历也有许多专业技术名词比较多，但是问到的时候回答出来的并不多！这里特别说明一点，在面试的时候，要认清自己的技术水平，要自信，但是不建议过于自信或者自负。另外，宁愿适当夸大些自己的技术水平，也不要学历，项目经历，工作经验作假，特别是学历和工作经验，只要一查出来，就没戏了！

    3.现在无良培训机构到处都是，虚张声势的宣传广告的满大街都是。希望大家多了解，咨询，学会鉴别，选择靠谱的一家培训机构，但是真正拿高薪，深入学习，还是要靠自己努力学习，钻研。

    4.也接触过几个在培训的人，在培训的过程中，他们自己非常努力深入的学习，钻研，做事很踏实，项目做得也不错，技术水平就不多说了。

    5.IT这一行，培训机构只是一个入门途径，不是捷径，要提高自己，拿高薪，在行业立足，还是要靠自己不断学习，不断实践！

    5.个人建议

    关于选择培训机构或者学习的建议，我简单的说一下，大家也可以到网上去了解。按照自己的需求，了解培训机构是否靠谱，那些培训机构靠谱等等的问题。

    1.评估自己是否需要去培训。

    如果对哪一门课程有兴趣，建议大家先在网上找下学习的资源（视频，文章，书籍等），关于初学入门学习的资源，我给出下面几个（下面有链接的地方）。大约半个月或者一个月左右的时间，评估下自己的自学的能力和自制力，如果觉得自己能学得进去，自己的学习能力和自制力又不是特别差，也有时间得情况下，个人觉得可以自学，自学的途径比如看视频，看文章，看书（我下面也附上了一点教程）等。不需要浪费钱去报培训班，毕竟培训机构也是说一些很基础的东西，甚至讲得没自学那么仔细，而且对于同样的知识点的记忆，自己自学的可能比在培训机构学的更加深刻！

    2.认清自己为什么去培训，是想高薪，还是有兴趣学习相关的课程。

    如果想去培训机构，建议对要学习的知识有一个大概的了解，哪怕是自己在家里看过几集视频或者十几页书，自学过几个月，这样一来可以让自己认识下是否对将要学的知识是否有兴趣，如果没兴趣，以后工作会很痛苦，路也不会走得太远。二来如果一点基础都没有在培训机构学习的过程中，可能会掉队，毕竟一个班那么多人，课程那么多，时间几个月，不可能因为你一个人耽误全班人的学习进度。

    3.选择去学习的培训机构，要谨慎考虑。

    要从多个方面（讲师水平，往期学员质量，口碑等）去对比，考虑。多咨询别人的看法和和建议，从而选择一个尽量自己觉得靠谱的培训机构！

    4.去培训机构千万别贷款

    需要贷款的培训机构一点都不靠谱，培训贷款的情况，银行一般不会放款，很多人贷款都是找第三方贷款机构。如果倒霉，遇上了利息高的贷款机构，这下还款的压力会很大，绝对很大。而且去培训的，一般都是一些应届生，或者转行的人，这些大多数经济都不太好的，加上自己也不知道培训机构是否靠谱，如果这样的情况，还去贷款培训的话，可能真的就是挖坑给自己跳。

    5.遇到不懂的时候，一定要弄懂，不然很容易掉队。

    遇到不懂的时候，一定要弄懂，不管什么途径，尽量不让自己掉队。如果真的掉队了，一定要抓紧跟上，实在是不懂的，一定要问老师，既然收钱了，老师就有义务回答！如果在培训机构都不敢问老师，那以后工作上又怎么敢问同事？

    6.多动手练习。

    即使是培训机构，也是一个指导的作用，随便看一个培训机构的课程清单，少则100多个知识课程，多则差不多300多个知识课程，几个月时间（一般是4个月）时间学习完。时间这么短，学这么多，想必大家那些课程都是教得很基础，要深入了解还是要靠自己学习，练习。就像我在上面说的：培训机构只是一个入门途径，不是捷径，要提高自己，拿高薪，在行业立足，还是要靠自己。不要以为只要进培训机构学习了，就一定能走上人生的巅峰，能让自己走上巅峰的只有自己，不是培训机构！

    7.与时俱进的学习。

    以web前端为例。在这几年，web前端的技术更新的非常快，我刚工作的时候还是jquery为王，现在是各种框架，库，工具，百花齐放，层出不穷。如果想在这行立足，就必须要不断学习。说到学习，我说一个建议就是，学习框架，库，工具之前，学好基础（html+css+js，附录下面附上我刚学习html+css+js的一点资料）。不要直接学框架，库等，从而忽略了基础！

    8.刚开始工作，不应该在在乎待遇，应该在乎收获。

    在刚开始工作的时候，有人带和没人带是两个完全不同的概念，有人带的话，能更快的学到更多的东西，少走很多弯路，收获自然会更多，所以，刚开始工作工资可以稍低一些，但是自己一定要注意公司或者同事能不能让你更好的成长！不建议大家像我之前接触到的那些人一样（基础没打牢，结果各种精通，自带两三年工作经验的人）。
    这样很容易造成下面的情况：
    1.如果是技术人员面试，很容易穿帮，到时候可能会很尴尬，很丢脸。
    2.容易给一个自负，不诚信的不好印象。
    3.不利于自己找工作和成长！

    6.附录

    下面简单推荐一些学习的资源，无论自己是什么情况，都建议看一下下面的学习资源，了解下自己。

    如果看这些资源，觉得没兴趣学下去，可能不适合接触这个行业。

    如果看这些资源，觉得自己有兴趣学，而且觉得能学进去，能掌握当中的知识，可能不需要去培训机构。

    如果看这些资源，觉得有兴趣，但是学的吃力，当中的知识掌握不了的话。可能去培训机构也是一个好的选择，但是谨慎选择，因为现在的行情，感觉是不靠谱的培训机构居多，所以大家要擦亮眼睛选一个靠谱的！

    关于培训机构，就说这么多了。至于大家选择哪一条路，自己做决定，选择认为一条最适合自己的路！但是，无论选择那一条路，要想学到东西，提升技术水平，在行业立足，最主要的还是要靠自己努力的学习！无论是什么出身，水平，只要从事了web前端甚至是IT这个行业，希望大家都保持学习的欲望，与时俱进的学习！这样才能在行业立足，不会被淘汰。

    7.小结

    我个人对培训机构的看法，大概就到这里了。至于培训机构是否靠谱，是否建议去学习，这里我就不多说了，意见也保持中立！但是大家去学习的时候，切记要谨慎考虑，选择！最后，无论是否选择去培训机构学习，要想深入学习或者成长，在行业立足，都是要靠自己努力学习，最重要的是多练习，这样才能有更多的收获，更好的成长！
