    特点

    • 不需要配置；
    • 对 React, JSX, ES6 和 Flow 可以直接编译；
    • 开发服务器；
    • 浏览器热加载的功能；
    • JavaScript 文件中可以直接 import CSS 和图片；
    • 自动处理 CSS 的兼容问题，无需添加 -webkit 前缀；
    • 集成好了编译命令，编译后直接发布成产品，包含 sourcemaps。
