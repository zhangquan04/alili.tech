让Visual Studio Code按照ESLint规则格式化你的代码

2017-11-05
0
React

项目加入ESLint之后,大家的代码是工整了许多.
但是因为刚刚不熟悉ESLint的规则,很多时候都在数进退多少空格,双引号还是单引号的删改切换.

严重影响了开发效率与心情!!!!

重要的事情说三遍!!!

下面介绍一种利用Visual Studio Code ESLint插件,配置之后,可以自动格式化成ESLint规定的格式

  1. 在Visual Studio Code 安装 ESLint插件
  2. widnows 点击 文件>首选项>设置 / mac 点击Code>首选项
  3. 搜索 eslint.autoFixOnSave, 找到之后修改为 true

重启你的Visual Studio Code,当你打开代码文件的后,直接保存文件.
代码会自动格式化成ESLint规定的格式.大大的提高了开发效率,并且统一了代码规范
